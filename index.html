About Baker Hub

cathcoach4u/personal-baker-hub
Personal family finance, health & household management app
Built With
💻
GitHub
Source code hosting & version control
🤖
Claude (Anthropic AI)
AI-assisted development & code generation
⚡
Supabase
Backend database, auth & real-time data
Stack
  • Vanilla HTML / CSS / JavaScript (no frameworks)
  • Supabase for database & storage
  • GitHub Pages for hosting
  • Claude Code for AI-assisted development
  • PWA manifest for home screen install
Features
  • Dashboard — Weekly reminders (bins, cleaners/Fiona), bulky waste countdown, NDIS funding overview, upcoming premiums
  • NDIS — Transaction tracking with provider dropdown, auto-fill, receipt/invoice uploads, edit in place
  • Insurance — Policies grouped by owner (collapsible), add/edit forms, monthly & annual totals
  • Contacts — Family contacts with categories, person tags (Family, Sarah, Cath, Andrew, Russell), tap to edit
  • House — Project tracking with priority & status, due dates
  • Dates — Important dates with council info, upcoming alerts, location & time details
Supabase Tables
  • ndis_funds — Funding bucket budgets
  • ndis_transactions — NDIS transaction history with receipt uploads
  • ndis_providers — Provider names with default NDIS numbers & funds
  • insurance_policies — Insurance & superannuation policies
  • contacts — Family contacts with categories & person tags
  • house_projects — House project tracking
  • important_dates — Important dates & council events
